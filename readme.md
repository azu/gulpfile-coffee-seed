# gulpfile-coffee-seed

Example of `gulpfile.coffee`.

## Installation

``` sh
$ npm install
```

## Usage

``` sh
$ gulp --require coffee-script
```

``--require`` is a [Compilers](https://github.com/gulpjs/gulp#compilers "Compilers") option.
